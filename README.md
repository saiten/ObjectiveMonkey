# ObjectiveMonkey

Hot patch for iOS App

## Requirements

- iOS 8 or Later

## Installation

ObjectiveMonkey is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "ObjectiveMonkey"
```
